# Запуск проекта

- `npm i`
- `npm run filldb 20`
- `npm start`

---

## Файл `.env`

- `DB_HOST` — домен для подключения к БД
- `DB_PORT` — порт для подключения к БД
- `DB_NAME` — название базы данных
- `DB_USER` — имя пользователя
- `DB_PASSWORD` — пароль
- `SESSION_SECRET`— секретное слово
